Quick Start (2 Minutes)

  1. Run backend on http://localhost:3000.
  2. Run frontend on a different port: npx serve frontend -l 3001
  3. Set Backend Base URL below to http://localhost:3000.
  4. Click Run Demo Flow or use form dummy-data buttons.

Set backend URL and save. If you see 404, check ports.
